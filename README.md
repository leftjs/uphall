# uphall

## The api for UpHall (尚食堂)



## To Contribute

###  init first:

``` bash
$ git clone git@github.com:leftjs/uphall.git
$ cd uphall/
$ npm install -g gulp bower
$ npm install 
$ bower install 
$ gulp
```
